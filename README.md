# CMS - Sistema de Gestión de Contenidos

Proyecto académico desarrollado con React, Node.js, Express y MySQL.

## Stack
- Frontend: React + TailwindCSS → Vercel
- Backend: Node.js + Express + Sequelize → Railway
- Base de datos: MySQL → Railway

## Desarrollo local

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Edita .env con tus datos de MySQL
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm start
```

Usuario admin por defecto: `admin@cms.com` / `Admin1234!`

## Despliegue
- Backend + MySQL: Railway
- Frontend: Vercel
